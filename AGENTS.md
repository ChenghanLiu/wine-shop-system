# Project Instructions for Codex

Project: Wine Shop System (Graduation Project)

Tech Stack:
- Backend: Spring Boot + MyBatis + MySQL
- Frontend: Vue.js + Element UI
- API: RESTful

System Type:
E-commerce web application for selling alcoholic beverages.

Roles:
1. User
2. Admin

User Features:
- Register / Login
- Browse products
- Search products
- Product categories
- Shopping cart
- Place orders
- Manage addresses
- Favorite products
- Comment on orders
- View activities
- View announcements

Admin Features:
- Manage users
- Manage products
- Manage categories
- Manage orders
- Manage comments
- Manage banners
- Manage activities
- Sales statistics

Product Categories:

Primary Categories:
- 白酒
- 啤酒
- 红酒
- 洋酒

Secondary Categories:

白酒：
酱香、浓香、清香、兼香、其他香型

啤酒：
工业啤酒、精酿啤酒、进口啤酒、果味啤酒

红酒：
干红、甜红、起泡酒、进口红酒、国产红酒

洋酒：
威士忌、白兰地、伏特加、朗姆酒、金酒

Goal:
Build a clean and visually appealing demo system suitable for a graduation project defense.
